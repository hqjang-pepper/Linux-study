# 201229 TIL(1)

### cd
`cd` : 작업 디렉토리를 홈 디렉토리로 변경
`cd -` : 작업 디렉토리를 이전 디렉토리로 변경

### ls
- 한번에 여러 디렉토리 목록 확인 방법 : `ls a b` : a 디렉토리와 b 디렉토리 목록 한번에 확인 가능  


| option  |   기능|
|---|---|---|---|---|
| `- a`  | .으로 시작하는 숨김파일 포함 모든 파일 보기  |  
|   `- d`|  디렉토리 정보를 자세하게 확인 가능  |   
|  `- l` |  좀더 자세한 정보 출력 |   
| `- r`  |  출력 결과 역순 |  
|   `- s`|  크기순으로 정렬 |   
|  `- t` |  수정 시간순으로 정렬 |
